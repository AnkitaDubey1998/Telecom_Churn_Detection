About Project

  • This customer churn detection project is a mini project implemented using the Telecom Customer Churn dataset available here.
  • The user has to give certain details about the customer the the system will predict whether the customer may stop using the service or not.
  • Gradient Boosting algorithm is used for classification of churn.
  • The source code of the project is available in my GitHub repository here.
